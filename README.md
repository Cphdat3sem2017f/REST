## Learning objectives

ORM / JPA / JPQL
- Create database connection
- Create persistence unit
- Use annotations
- Create entities
- Handle relations
- Handle inheritance
- Handle entities ( Persist / Merge / Remove / Find )
- Use entity managers
- Create named / dynamic queries
- Create facade to handle entities
